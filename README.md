# 非同期 JavaScript

このリポジトリは、非同期JavaScriptに関しての備忘録ですが、index.htmlファイルはレスポンシブWebデザインを適用しています。ディスプレイの幅が799px以下のデバイスでサイトを閲覧した場合、交差監視API（Intersection Observer API）に関するCSSの挙動は機能しないソースコードとなっています。
