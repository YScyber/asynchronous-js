JSON

JavaScript Object Notation（JSON、読み方はジェイソン）はウェブアプリケーションで使用するデータ転送言語の１つです。JavaScriptのオブジェクト構文に基づいてテキストベースの形式で構造化されたデータを表現しています。

サンプルのJSONファイルを３つ作成しました。内容は以下のとおりです。

  • cats.json
    • [ 配列 ]の中に{ オブジェクト }があり、データ（"name" : "名前", "breed" : "猫種", "gender" : "性別"）が６種類入っています。
  • dogs.json
    • [ 配列 ]の中に{ オブジェクト }があり、データ（"breed" : "犬種", "image" : "画像のURL", "features" : "特徴"）が８種類入っています。
  • heroes.json
    • [ 配列 ]の中に{ オブジェクト }があり、データ（"name" : "名前", "age" : "年齢", "realName" : "本名", "technique" : [ "技名" ]）が７種類入っています。
    • [ "技名" ]は配列となっており、いくつかの技名が入っています。
